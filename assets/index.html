    HOW TO USE

    1. Type digits on the keypad, then press an operation key (+ − × ÷) to run it against A.
    2. Press a register key (F E D C M B) first to use that register as the operand instead.
    3. ◇ prints to the tape, ✳ clears, S acknowledges an error or stops a run.
    4. Load a card from the CARDS list, then press V / W / Y / Z to run it.
    5. REC records a new program; mark start points with V=1 W=2 Y=3 Z=4, then save it with a title.
